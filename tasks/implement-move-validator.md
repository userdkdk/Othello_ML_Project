# Move Validator 구현 작업 지시서

## 목적
현재 플레이어 기준 유효 수 판정과 뒤집기 대상 계산을 구현한다.

## 선행 입력
- `docs/othello-rules.md`
- `docs/othello-acceptance-tests.md`
- `docs/othello-module-spec.md`
- `agents/engine-agent.md`

## 필수 작업
- 단일 좌표 유효성 검사 구현
- 뒤집히는 좌표 목록 계산 구현
- 뒤집히는 방향 정보 계산 구현
- 현재 플레이어의 전체 유효 수 목록 계산 구현
- 8방향 스캔 로직 테스트 작성

## 완료 조건
- 초기 상태 유효 수 4개를 정확히 계산한다.
- 초기 상태 `WHITE` 유효 수 4개를 정확히 계산한다.
- 다중 방향 뒤집기 케이스를 통과한다.
- 이미 돌이 있는 칸, 범위 밖 좌표, 뒤집을 수 없는 빈 칸을 모두 유효하지 않은 수로 판정한다.
- 유효하지 않은 좌표에서 상태 변경 없이 실패한다.
