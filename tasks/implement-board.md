# Board 구현 작업 지시서

## 목적
오셀로 보드 표현과 좌표/칸 상태 조작 기능을 구현한다.

## 선행 입력
- `docs/othello-rules.md`
- `docs/othello-module-spec.md`
- `agents/engine-agent.md`

## 필수 작업
- 8x8 보드 상태 표현 정의
- 좌표 범위 검증 구현
- 칸 조회/갱신 API 구현
- 초기 보드 생성 기능 구현
- 돌 개수 확인 기능 또는 보조 함수 구현
- 보드 직렬화 또는 스냅샷 추출 기능 구현

## 완료 조건
- 초기 상태를 정확히 표현할 수 있다.
- 범위를 벗어난 좌표를 안전하게 처리한다.
- 상위 모듈이 보드 복사 또는 불변 처리 전략을 선택할 수 있다.
- 최소한 보드 64칸 상태를 재구성할 수 있는 직렬화 표현을 제공한다.
