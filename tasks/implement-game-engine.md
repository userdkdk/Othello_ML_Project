# Game Engine 구현 작업 지시서

## 목적
보드, 유효 수 계산, 턴 전환, 패스, 종료 판정을 포함한 전체 게임 진행 로직을 구현한다.

## 선행 입력
- `docs/othello-rules.md`
- `docs/othello-acceptance-tests.md`
- `docs/othello-module-spec.md`
- `agents/engine-agent.md`

## 필수 작업
- 새 게임 생성
- 수 적용
- 패스 처리
- 종료 여부 판정
- 승패 계산
- 실패 시 상태 불변 보장
- 종료 후 추가 `move`와 추가 `pass` 차단

## 완료 조건
- 인수 테스트 문서의 초기 상태, 수 적용, 패스, 종료 시나리오를 재현할 수 있다.
- 종료된 게임에서 추가 수 적용과 추가 패스를 모두 차단한다.
- 실패 시 보드, 현재 플레이어, 종료 여부, 승패 결과, 부가 상태가 유지된다.
- 리뷰 가능한 수준의 테스트가 함께 준비되어 있다.
