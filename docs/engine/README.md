# Engine Docs

## 읽는 순서
1. `othello-rules.md`
2. `othello-module-spec.md`
3. `othello-acceptance-tests.md`

## 파일 역할
- `othello-rules.md`
  - 실제 오셀로 규칙과 종료/패스 기준
- `othello-module-spec.md`
  - `board`, `move_validator`, `game_engine` 경계
- `othello-acceptance-tests.md`
  - 엔진이 통과해야 하는 인수 시나리오
